# KNN Assignment

This project implements the K-Nearest Neighbor (KNN) algorithm from scratch using Python.

## Features
- KNN Classification using Iris Dataset
- KNN Regression using Diabetes Dataset
- Train, Validation, and Test Split
- Euclidean Distance Calculation
- Accuracy and Mean Squared Error Evaluation

## Technologies Used
- Python
- Jupyter Notebook

## Datasets
- Iris Dataset
- Diabetes Dataset
